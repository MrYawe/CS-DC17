# CS-DC17

## Compilation .ez file
You will need to have EASEA installed : http://easea.unistra.fr/index.php/EASEA_platform <br />
```shell
easea csdc17.ez
```


## Compilation C++ file
```shell
make
```

## Execute csdc17 
```shell
./csdc17
```
